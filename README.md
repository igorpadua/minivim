# Minivim

Este programa é um vimLike feito com C++, tendo como base o canal TerminalRoot.

## Funcionalidades

* É possível abrir um arquivo;
* É possível savar um arquivo;
* "Normal Mode";
* "Insert Mode";
* Sair do programa.

## Requisitos

* CMake
* C++
* ncurses

## Executar

> mkdir build
> 
> cd build
> 
> cmake ..
> 
> make
> 
> ./minivim
